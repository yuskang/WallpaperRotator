## Q

如何把.jxl轉成.png

## A

考量到您只有兩張圖片（直向/橫向）需要轉檔以放入 Q7311 使用，且根據您的背景（熟悉 Mac 終端機操作），我有以下三種建議，依照推薦程度排序：

### 方法一：使用 Squoosh (最快，免安裝，Google 開發)

如果您不想為了轉兩張圖安裝軟體，這是最佳解。它是 Google Chrome Labs 開發的工具，運算是在本機瀏覽器執行（Local），隱私性高且支援 JXL。

1.  開啟瀏覽器（Mac 或 Q7311 皆可）前往 **[squoosh.app](https://squoosh.app/)**。
    
2.  將您的 `.jxl` 檔案拖入視窗。
    
3.  右下角的 "Compress" 選單中，選擇 **PNG** (若要無損) 或 **MozJPEG** (若要一般圖片)。
    
4.  按下右下角的下載按鈕即可。
    

* * *

### 方法二：在您的 Hackintosh 使用終端機 (最適合您的習慣)

既然您習慣使用 Homebrew 和 zsh，這是最「工程師風格」的快速解法。利用 `imagemagick` 強大的轉檔功能。

1.  打開您的 Mac 終端機 (Terminal/iTerm2)。
    
2.  若尚未安裝 ImageMagick，請輸入：
    
    ```
    brew install imagemagick
    
    ```
    
3.  執行轉檔指令：
    
    ```
    # 單張轉換
    magick input.jxl output.png
    
    # 或者進入資料夾，一次把所有 jxl 轉成 png
    mogrify -format png *.jxl
    
    ```
    
4.  轉好的檔案再透過 AirDrop 或雲端傳給 Q7311。
    

* * *

### 方法三：在 Q7311 上安裝軟體 (XnView MP)

如果您希望這台 Fujitsu 平板本身具備看圖與轉檔能力，**XnView MP** 是目前對 JXL 支援度最好的免費看圖軟體（比 IrfanView 方便，因為 IrfanView 還需要另外裝 Plugin 才能讀 JXL）。

1.  下載並安裝 [XnView MP](https://www.xnview.com/en/xnviewmp/)。
    
2.  用它開啟 .jxl 檔案。
    
3.  按 `Ctrl + Shift + S` (另存新檔)，選擇 PNG 格式儲存。
    

### 下一步建議

轉檔完成後，記得檢查圖片解析度是否符合 Q7311 的螢幕（通常是 **1920x1080**），如果比例不對，可能會在自動切換時造成黑邊或拉伸。
