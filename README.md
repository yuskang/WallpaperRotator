[English](#english) | [繁體中文](#繁體中文)

## English

Windows tablet wallpaper auto-switcher based on screen orientation.

### Features

- **Real-time Detection** - WMI event-driven screen orientation monitoring
- **Auto Switch** - Automatically switches wallpaper between landscape/portrait
- **Perfect Fit** - Supports Fit/Fill/Stretch display modes
- **Background Running** - Runs silently in system tray
- **Power Efficient** - Event-driven design, near-zero CPU usage

### Compatible Devices

- Fujitsu Q7311 / Q738
- Microsoft Surface series
- Other Windows 10/11 tablets

### System Requirements

- Windows 10 1903+ or Windows 11
- .NET 8.0 Runtime (included in self-contained build)

### Installation

Download the latest version from [Releases](../../releases).

- **WallpaperRotator-x64.zip** - For Intel/AMD processors
- **WallpaperRotator-arm64.zip** - For ARM processors (Surface Pro X, etc.)

### Usage

1. Run `WallpaperRotator.exe`
2. Right-click the system tray icon to open settings
3. Set your landscape and portrait wallpapers
4. Enable "Start with Windows" for automatic startup

## 繁體中文

Windows 平板螢幕旋轉自動切換桌布工具

### 功能特色

- **即時偵測** - 使用 WMI 事件驅動偵測螢幕方向變化
- **自動換圖** - 橫向/直向自動切換對應桌布
- **完美適配** - 支援 Fit/Fill/Stretch 等多種顯示模式
- **背景執行** - 系統托盤運行，完全無感
- **省電設計** - 事件驅動，CPU 佔用近乎為 0

### 適用設備

- Fujitsu Q7311 / Q738
- Microsoft Surface 系列
- 其他 Windows 10/11 平板電腦

### 系統需求

- Windows 10 1903+ 或 Windows 11
- .NET 8.0 Runtime（獨立發布版已內含）

### 安裝

從 [Releases](../../releases) 下載最新版本。

- **WallpaperRotator-x64.zip** - 適用於 Intel/AMD 處理器
- **WallpaperRotator-arm64.zip** - 適用於 ARM 處理器（Surface Pro X 等）

### 使用方式

1. 執行 `WallpaperRotator.exe`
2. 右鍵點擊系統托盤圖示開啟設定
3. 設定橫向與直向桌布
4. 啟用「開機自動啟動」功能
